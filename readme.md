# 🌱 Crop Recommendation System

A Machine Learning-based Crop Recommendation System that predicts the most suitable crop based on soil nutrients and environmental conditions. The project uses a trained ML model and a Streamlit web application to provide instant crop recommendations.

---

## 📌 Project Overview

Agriculture plays a vital role in food production, and choosing the right crop is essential for maximizing yield. This project recommends the most suitable crop by analyzing soil nutrients and weather conditions.

The system takes the following inputs:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature (°C)
- Humidity (%)
- Soil pH
- Rainfall (mm)

Based on these inputs, the machine learning model predicts the most appropriate crop for cultivation.

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Crop-Recommendation-System.git
```

### 2. Move to the project directory

```bash
cd Crop-Recommendation-System
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit app

```bash
streamlit run app.py
```
